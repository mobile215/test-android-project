How to build GoogleAccountUtil.apk

Getting the Source:
-------------------
repo init -u persistent-https://googleplex-android.git.corp.google.com/platform/manifest -b tradefed
repo sync

Building:
---------
. build/envsetup
tapas GoogleAccountUtil
make
